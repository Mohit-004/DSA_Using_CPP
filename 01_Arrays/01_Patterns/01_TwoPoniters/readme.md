# ⚠️ Common Mistakes

- [ ] Infinite loop
- [ ] Forgot to move a pointer
- [ ] Pointer out of bounds
- [ ] Wrong loop condition (`left < right` vs `left <= right`)
- [ ] Incorrect duplicate skipping
- [ ] Swapping with the wrong pointer
- [ ] Forgetting to sort before applying two pointers
- [ ] Returning the wrong pointer (`left`, `right`, or `left + 1`)
